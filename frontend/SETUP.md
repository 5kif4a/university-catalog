# Инструкция по настройке проекта

## Выполните следующие команды

Откройте терминал в директории `/Users/alikhan_shorin/Documents/personal/ai-project/frontend` и выполните команды:

### 1. Установка зависимостей

```bash
cd /Users/alikhan_shorin/Documents/personal/ai-project/frontend
yarn install
```

Эта команда установит все зависимости:
- React 19
- TypeScript
- Vite
- MUI (Material UI) v7
- TanStack Router
- Tailwind CSS v4
- И все необходимые dev-зависимости

### 2. Запуск dev сервера

```bash
yarn dev
```

После запуска приложение будет доступно по адресу: `http://localhost:5173`

### 3. Проверка работы

Откройте браузер и перейдите на `http://localhost:5173`. Вы должны увидеть:
- Главную страницу с приветствием
- Навигационное меню (Home, About, Users)
- 3 карточки с описанием возможностей

Проверьте навигацию:
- Кликните "About" - откроется страница с технологиями
- Кликните "Users" - откроется страница с моковыми пользователями

### 4. Сборка для продакшена (опционально)

```bash
yarn build
```

Собранные файлы будут в директории `dist/`

### 5. Предпросмотр продакшн сборки (опционально)

```bash
yarn preview
```

## Что уже настроено

1. **React 19 + TypeScript** - современный стек с полной типизацией
2. **Vite** - быстрая сборка и HMR
3. **TanStack Router** - type-safe роутинг с автогенерацией типов
4. **MUI v7** - современная тема без градиентов в минималистичном стиле
5. **Tailwind CSS v4** - настроен для работы с MUI (preflight отключен)
6. **Path aliases** - можно использовать `@/` для импорта из `src/`

## Что создано

### Страницы
- **Home** (`/`) - главная страница с описанием проекта
- **About** (`/about`) - страница с информацией о технологиях
- **Users** (`/users`) - пример страницы с моковыми данными

### Компоненты
- **Layout** - обертка с навигацией и контейнером
- **Navigation** - навигационное меню
- **UserCard** - карточка пользователя

### Тема
- Современный минималистичный дизайн
- Цветовая схема: синий primary, серый secondary
- Без градиентов (как требовалось)
- Тени в стиле Tailwind
- Скругленные углы

## Следующие шаги

Проект готов к разработке! Вы можете:
1. Добавлять новые страницы в `src/pages/`
2. Создавать роуты в `src/routes/`
3. Добавлять компоненты в `src/components/`
4. Позже подключить TanStack Query для API
5. Добавить тесты

## Возможные проблемы

Если при запуске возникают ошибки:

1. **Ошибка с @types/node**: Уже добавлен в devDependencies
2. **Ошибка с routeTree.gen.ts**: Файл генерируется автоматически при первом запуске
3. **Ошибка с Tailwind**: Убедитесь что установлена версия 4.x

Если проблемы сохраняются, попробуйте:
```bash
rm -rf node_modules yarn.lock
yarn install
```
